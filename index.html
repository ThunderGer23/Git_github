► Clase 1

Introducción a la Web

Siendo el internet una herramienta fundamental hoy día, es necesario conocer ampliamente su funcionamiento, tanto como un todo, así como los componentes que lo conforman.


Los sítios web funcionan a partir del modelo cliente-servidor, es decir, un dispositivo final conectado a un servidor para obtener información de lo que conocemos como internet.


Un cliente, como ya se dijo antes, es un dispositivo que solicita información y una vez teniéndola la consume según esté programado. El cliente usualmente es el que tiene contacto directo con el usuario, y está programado para trabajar de forma local.


Un servidor es un dispositivo que guarda toda la información de un servicio, mismo que presta a varios clientes a través de internet, ya sea de almacenamiento o procesamiento.


Tanto cliente como servidor se comunican a través del uso de direcciones IP, sin embargo, para comunicarse con un servidor, un cliente normalmente utiliza un servicio DNS, que asocia la dirección IP del servidor con un nombre más fácil de recordar, como por ejemplo www.google.com que es más fácil de recordar que una dirección tipo 254.254.254.254.


Al momento que realizamos una conexión con un sitio web a través de este método, lo que nos devuelve el sitio web es la página de inicio de dicho sitio, generalmente creada en un archivo llamado "index.html".


Este sitio, como su extensión lo indica, está creado en HTML, que es un lenguaje de etiquetas por el cuál se carga el contenido visual de un sitio web. Cabe destacar que HTML es un lenguaje estructurado, o sea, que genera el contenido en el orden en el que está escrito, y conforme el explorador de internet lo va interpretando.


IMPORTANTE: HTML NO ES UN LENGUAJE DE PROGRAMACIÓN!

► Clase 2

Elementos de bloque en HTML

En general, los elementos de bloque se definen como elementos que, sin importar su contenido, ocupan totalmente el espacio horizontal de la pantalla.


Cuando hablamos de elementos de bloque en HTML podemos mencionar 2 elementos muy importantes al momento de administrar la información.


Estos son Article y Section, y funcionan de la manera mencionada a continuación.

Section

La etiqueta <section> funciona como un contenedor genérico que agrupa contenido relacionado entre sí. Cuando se necesita agrupar contenido en un sólo bloque total es que se utiliza section.

Article

A diferencia de <section>, <article> sirve para agrupar contenido que tiene contexto propio, de manera que no importa desde dónde se cite, es contenido claro y de facil comprensión.

Elementos de Linea

Se consideran elementos de linea a aquellos que se manejan sólamente en una linea del código HTML, y que ocupan sólamente el espacio que estos requieren.


El más notable de estos es el salto de línea, que se llama con la etiqueta <br>, y, como su nombre lo indica, da un salto a la siguiente linea.

Elementos de Linea de Ponderación

Así mismo, tenemos elementos de linea de Ponderación, que se encargan de dar prioridad a un elemento sobre otro al momento de presentarlos en pantalla.


Los más conocidos de estos son las etiquetas <em> y <strong>. Cabe aclarar que la etiqueta <em> tiene una prioridad mayor que la etiqueta <strong>.

► Clase 4

Enlace

Los enlaces o links sirven para conectar una página web con otra, y se pueden utilizar como un recurso tanto interno como externo.


Llevan el atributo obligatorio href, en el cuál se especifica la ruta que se va a utilizar.


Si ambos documentos están en la misma ubicación, basta con colocar el nombre del archivo, de lo contrario habrá que especificar la ruta completa.

Su sintáxis es:

<a href="http://ww.url.com">Texto</a>


Existen 2 tipos de rutas, que son las siguientes:

► Relativas:
  Pueden ser relativas al punto donde nos encontramos o a la raíz del proyecto.

► Absolutas:
  Tienen un protocolo http o https y son únicas en la red. se suelen utiliar para rutas ajenas al sistema.

Target

Se coloca de manera inmediata después del href, de la siguiente forma:

<a href="" target="_blank"> </a>

y su principal uso es definir el modo de apertura del recurso solcitado. el valor sólo puede ser _self o _blank, siendo _self el valor por defecto.

Download

Es un atributo tipo booleano, sirve para poder descargar el recurso solicitado. Dicho recurso debe encontrarse en nuestro servidor para poder ejecutarlo. su sintáxis es

<a href="" download> </a>

Saltos entre etiquetas

A demás de llevar a otras páginas, href también puede llevarnos a otros lugars dentro dde la misma página, a través del uso de un id.

<a href="#id">Texto</a>

► Clase 5

Listas

Las listas en HTML sirven para enumerar contenido, con o sin un órden específico. existen 3 tipos de etiquetas que se pueden utilizar en la creación de listas en HTML, y son las siguientes:

<ul> - unordered list

Vienen en desorden los objetos listados, y pueden ser enumerados con viñetas de distintos estilos a través del atributo style, con el subatributo list-style-type:
  • disc
  • circle
  • square
  • 👀 none

sintáxis: 
                    

                      
<ul style:"list-style-type:none">
                      
  <li>Objeto 1</li>
                      
  <li>Objeto 2</li>
                      
</ul>

<ol> - ordered list

Es una lista que permite ver los objetos de forma ordenada, y puede ordenarse a través de números ordinarios, letras y números romanos a través del atributo type:
  1. Numerados
  2. type="1"
  1. Letras mayúsculas
  2. type="A"
  1. Letras minúsculas
  2. type="a"
  1. Números romanos (Mayús)
  2. type="I"
  1. Números romanos (Minús)
  2. type="i"

Sintáxis: 
                  

                    
<ol>
                    
  <li>Objeto 1</li>
                    
  <li>Objeto 2</li>
                    
</ol>

<dl> - definition list


Permite crear una lista de pares, con palabras clave contra definiciones.
Sintáxis: 


                  

                    
<dl>
                    
  <dt>Objeto 1</dt>
                    
  <dd>Descripción</dd>
                    
  <dt>Objeto 2</dt>
                    
  <dd>Descripción</dd>
                    
</dl>

Y se vería de esta forma:
  
Objeto 1
  
Descripción
  
Objeto 2
  
Descripción

Tablas

Sirven para mostrar contenido tabulado. la estructura básica se compone de:

<table> - para encerrar la tabla
<tr> - table row - fila
<td> - table data - celda el numero de celdas en una tabla define el número de columnas.

Sintáxis: 
                  

                    <table style="border: 1px solid black;">
                      <caption>Nombre de la Tabla</caption>
                      <thead>
                        <tr>
                          <th></th>
                          <th>Columna 1</th>
                          <th>Columna 2</th>
                          <th>Columna 3</th>
                          <th>Columna 4</th>
                          <th>Columna 5</th>
                        </tr>
                      </thead>
                      <tbody>
                        <tr>
                          <td>Fila 1</td>
                          <td>1</td>
                          <td>2</td>
                          <td>3</td>
                          <td>4</td>
                          <td>5</td>
                        </tr>
                        <tr>
                          <td>Fila 2</td>
                          <td>4</td>
                          <td>3</td>
                          <td>2</td>
                        </tr>
                      </tbody>
                      <tfoot>
                        <tr>
                          <td>Fila Final</td>
                          <td>No hé</td>
                          <td>xD</td>
                        </tr>
                      </tfoot>
                    </table>

Y se vería de la siguiente forma:

Nombre de la Tabla
Columna 1 Columna 2 Columna 3 Columna 4 Columna 5
Fila 1 1 2 3 4 5
Fila 2 4 3 2
Fila Final No hé xD

Más etiquetas de bloque

► <address>
sirve para anotar información de contacto al article dentro del que se encuentre
► <blockquote>
Marca citas de otros autores o documentos. también permite poner un enlace al doc original
► <pre>
se utiliza para tener codigo preformateado que necesita ser representado igual que como se escribió
► <div>
se utiliza como división del documento, es un contenedor generico
► <hr>
sirve para cambiar de tema en el html
► <span>
es un contenedor de linea, equivale a div y sirve para encerrar palabras o pequeños textos para su uso con css y js
► <q>
equivalente a blockquote pero para una sola linea
► <code>
encierra código que se quiera represrntar, suele ir unido con la etiqueta pre.
