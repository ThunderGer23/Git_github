► Clase 1

Introducción a la Web

Siendo el internet una herramienta fundamental hoy día, es necesario conocer ampliamente su funcionamiento, tanto como un todo, así como los componentes que lo conforman.


Los sítios web funcionan a partir del modelo cliente-servidor, es decir, un dispositivo final conectado a un servidor para obtener información de lo que conocemos como internet.


Un cliente, como ya se dijo antes, es un dispositivo que solicita información y una vez teniéndola la consume según esté programado. El cliente usualmente es el que tiene contacto directo con el usuario, y está programado para trabajar de forma local.


Un servidor es un dispositivo que guarda toda la información de un servicio, mismo que presta a varios clientes a través de internet, ya sea de almacenamiento o procesamiento.


Tanto cliente como servidor se comunican a través del uso de direcciones IP, sin embargo, para comunicarse con un servidor, un cliente normalmente utiliza un servicio DNS, que asocia la dirección IP del servidor con un nombre más fácil de recordar, como por ejemplo www.google.com que es más fácil de recordar que una dirección tipo 254.254.254.254.


Al momento que realizamos una conexión con un sitio web a través de este método, lo que nos devuelve el sitio web es la página de inicio de dicho sitio, generalmente creada en un archivo llamado "index.html".


Este sitio, como su extensión lo indica, está creado en HTML, que es un lenguaje de etiquetas por el cuál se carga el contenido visual de un sitio web. Cabe destacar que HTML es un lenguaje estructurado, o sea, que genera el contenido en el orden en el que está escrito, y conforme el explorador de internet lo va interpretando.


IMPORTANTE: HTML NO ES UN LENGUAJE DE PROGRAMACIÓN!

 


 

► Clase 2

Elementos de bloque en HTML

En general, los elementos de bloque se definen como elementos que, sin importar su contenido, ocupan totalmente el espacio horizontal de la pantalla.


Cuando hablamos de elementos de bloque en HTML podemos mencionar 2 elementos muy importantes al momento de administrar la información.


Estos son Article y Section, y funcionan de la manera mencionada a continuación.

Section

La etiqueta <section> funciona como un contenedor genérico que agrupa contenido relacionado entre sí. Cuando se necesita agrupar contenido en un sólo bloque total es que se utiliza section.

Article

A diferencia de <section>, <article> sirve para agrupar contenido que tiene contexto propio, de manera que no importa desde dónde se cite, es contenido claro y de facil comprensión.

Elementos de Linea

Se consideran elementos de linea a aquellos que se manejan sólamente en una linea del código HTML, y que ocupan sólamente el espacio que estos requieren.


El más notable de estos es el salto de línea, que se llama con la etiqueta <br>, y, como su nombre lo indica, da un salto a la siguiente linea.

Elementos de Linea de Ponderación

Así mismo, tenemos elementos de linea de Ponderación, que se encargan de dar prioridad a un elemento sobre otro al momento de presentarlos en pantalla.


Los más conocidos de estos son las etiquetas <em> y <strong>. Cabe aclarar que la etiqueta <em> tiene una prioridad mayor que la etiqueta <strong>.

Atributos

En términos generales existen 2 tipos de atributos, que son:


► Comunes:
  Su sintáxis es: <etiqueta atributo="valor">

► Booleanos:
